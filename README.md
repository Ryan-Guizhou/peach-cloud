# 技术选型

### 1、nacos

**Spring Cloud Nacos 相比于原生的nacos在性能上会低一些，但是针对于当前项目，SpringBoot+SpringCloudAlibaba的兼容性会好很多，而且接入更加便捷**

****

| 场景             | Spring Cloud Nacos | Nacos 原生 | 差异       |
| :--------------- | :----------------- | :--------- | :--------- |
| **应用启动时间** | 1200ms             | 900ms      | 原生快25%  |
| **配置首次加载** | 200ms              | 150ms      | 原生快25%  |
| **配置热更新**   | 50-100ms           | 10-20ms    | 原生快80%  |
| **内存占用**     | 基础+50MB          | 基础+30MB  | 原生省20MB |
| **CPU使用率**    | 中等               | 较低       | 原生更优   |
